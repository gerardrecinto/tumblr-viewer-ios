# Tumblr Photo Viewer

An iOS photo viewer built with Swift and UIKit that pulls image posts from the Tumblr API.

## Features

- Scrollable feed of Tumblr photo posts
- Tap any image to view it full-screen in a detail view
- Auto Layout for consistent layout across device sizes

## Tech Stack

| Layer | Technology |
|---|---|
| Language | Swift |
| UI | UIKit, Auto Layout |
| Networking | Tumblr API v2 |
| Dependencies | CocoaPods |

## Setup

```bash
git clone https://github.com/gerardrecinto/tumblrLab.git
cd tumblrLab
pod install
open Tumblr.xcworkspace
```

Add your Tumblr API key before building.
